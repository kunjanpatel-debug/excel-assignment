1. CORE RIBBON TABS & THEIR PURPOSES:

* Home Tab: 

  The default tab used for core data formatting, styling, and basic arrangements. It contains essential tools for clipboard actions (copy/paste), font customization, text alignment, number formatting (currency, percentages, decimals), and conditional formatting rules.

* Insert Tab: 

  Used to add external objects, visual elements, and structural layouts to the workbook. Key features include creating PivotTables, embedding shapes or illustrations, and generating data visualization charts (such as column, line, or pie graphs).

* Data Tab: 

  Built exclusively for managing, processing, and cleaning transactional datasets. It holds powerful infrastructure for external data connections (Get & Transform Data/Power Query), sorting and filtering configurations, Data Validation rules, and structural cleaning tools like 'Text to Columns'.


2. QUICK ACCESS TOOLBAR (QAT) TOOLS:

* Save (Ctrl + S):

  A one-click tool that instantly commits the current state of the workbook to local or cloud storage, preventing data loss from unexpected application closures or system power cuts.

* Undo (Ctrl + Z): 

  Reverses the most recent action, edit, or data entry sequence in chronological order, allowing quick correction of structural errors or accidental data overwrites.
